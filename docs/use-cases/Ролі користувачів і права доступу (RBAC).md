#### \[!!! ВАЖЛИВО !!!\] Політики доступу
**Принцип мінімальних прав (Least Privilege):** кожен користувач має тільки ті права, які необхідні для виконання його обов’язків.
**Розмежування обов’язків (Separation of Duties):**
> Абітурієнт **_не може_** перевіряти або редагувати документи інших.  
> Оператор **_не має права_** реєструвати договори чи формувати накази.  
> Адміністратор **_не бере участі_** у процесах прийому документів.
**Принцип єдиного відповідального (Single Accountability):** кожен ключовий процес має визначену роль, яка є остаточно відповідальною за результат.
